# Data Analytics Project

## Overview

This project showcases a complete data analytics pipeline, from raw data to actionable insights. It demonstrates skills in data processing, analysis, visualization, and storytelling.

**Key objectives:**
- Analyze and clean real-world data
- Extract insights using SQL and Python
- Build an interactive dashboard
- Present findings through reports and presentations

## Dataset

- **Source:** Internal Raw Data
- **Format:** CSV
- **Domain:** Retail / E-commerce Analytics

## Tech Stack

- **Python:** Pandas, NumPy
- **Database:** MySQL
- **Visualization:** Power BI
- **Presentation:** Claude.AI
- **Environment:** Jupyter Notebook

## Workflow

**1. Data Loading**
- Imported dataset using Pandas
- Checked structure, data types, and summary statistics

**2. Exploratory Data Analysis**
- Visualized distributions and relationships
- Identified trends and anomalies

**3. Data Cleaning**
- Handled missing values and duplicates
- Standardized formats and corrected inconsistencies

**4. SQL Analysis**
- Loaded cleaned data into MySQL
- Performed aggregations, joins, filtering, and grouping
- Derived key business insights

**5. Power BI Dashboard**
- Built an interactive dashboard with KPIs, trend analysis, category breakdowns, filters, and slicers

**6. Reporting and Presentation**
- Created a detailed report
- Built a presentation using Claude.AI

## Key Insights

**Revenue by Gender**
- Male customers generated significantly higher revenue compared to female customers.
- This shows strong revenue contribution from the male segment.
- The female segment represents a growth opportunity through targeted campaigns and product recommendations.

**Discount Behavior**
- Many discount users still spent above the average purchase amount.
- Discounts helped attract valuable customers, not only low-spending buyers.
- Discount strategies should be optimized to increase sales while protecting profit margins.

**Subscription Insights**
- Subscribers did not spend significantly more than non-subscribers.
- This suggests the current subscription model needs stronger benefits.
- Exclusive offers, free shipping, or loyalty rewards could improve subscription value.

**Customer Segmentation**
- Loyal customers formed the largest customer group.
- This indicates strong customer retention.
- Returning customers can be targeted with rewards to convert them into loyal customers.

**Age Group Revenue**
- Young Adults generated the highest revenue.
- Other age groups also contributed consistently.
- This shows a balanced customer base with low dependency on a single age segment.

## Business Impact

- Helped identify high-value customer segments
- Revealed growth opportunities in female customers and new customers
- Highlighted the need to improve subscription benefits
- Supported better discount and pricing decisions
- Helped create data-driven recommendations for marketing, retention, and revenue growth
