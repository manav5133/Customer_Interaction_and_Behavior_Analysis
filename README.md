Data Analytics Project
Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data and progressing through analysis, visualization, and reporting. It highlights practical skills in data cleaning, exploratory analysis, SQL querying, dashboard creation, and presentation.

The main objectives of this project are:

To analyze and clean real-world data
To extract insights using Python and SQL
To build an interactive dashboard using Power BI
To present findings through a structured report and presentation
